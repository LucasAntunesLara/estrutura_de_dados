Modelar a matriz curricular como um Grafo e gere uma sequência válida de conclusão das disciplinas.

Requisitos de Implementação

- Modelagem do Grafo:
- Implemente um Grafo Dirigido (DAG) usando Lista de Adjacência.
- Utilize as disciplinas do curso e seus pré-requisitos como dados.
  Resultados:
- O programa deve imprimir a lista sequencial completa e válida de disciplinas.
- O código deve detectar e reportar qualquer ciclo de dependências encontrado.
